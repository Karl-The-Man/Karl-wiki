# The Boring Company

Tunneling infrastructure company founded by [[Elon Musk]]. Serves as the primary case study for [[Thinking in Limits]] in the source material.

## Limit-Based Analysis

**Problem:** Tunneling costs ~$1 billion per mile (LA subway extension baseline).

**Key insight:** Cities are built in 3D, but roads are only 2D. Underground tunnels can extend to arbitrary depth (deepest mines far exceed tallest buildings), so there's no fundamental limit on tunnel capacity — only a cost problem.

**First-principles cost reduction:**

| Lever | Mechanism | Improvement |
|---|---|---|
| Diameter | Shrink from 26–28 ft to 12 ft (area drops 4x) | ~4–5x |
| Machine uptime | Continuous boring + reinforcing (vs. alternating) | 2x |
| Machine power | Push toward thermal/power limits | 2–5x |
| **Combined** | | **~8–40x** |

*(elon-musk-on-first-principles.pdf)*
