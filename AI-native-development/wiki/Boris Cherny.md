# Boris Cherny

Creator and engineering lead of **[[Claude Code]]** at **[[Anthropic]]**. Author of the first O'Reilly book on TypeScript. Former Meta engineer (7 years) where he led code quality across all of Meta (Instagram, Facebook, Messenger, WhatsApp, Reality Labs).

## Background

- Started freelancing as a teenager, later became an early Y Combinator participant (first hire at a YC medical software startup called Agile Diagnosis, ~2011–2012)
- Spent 7 years at Meta: worked on Facebook Groups, Instagram dev infra, and code quality
- At Instagram, was one of the top 2–3 most productive engineers by code volume
- Led Meta's "Better Engineering" program (Zuckerberg mandated every engineer spend 20% of time fixing tech debt)
- Found through causal analysis that code quality contributes "double digit percent" to engineering productivity
- Wrote the first O'Reilly TypeScript book; deeply involved in the TypeScript community

## At Anthropic

- Hired by Ben Mann (Anthropic co-founder)
- His first PR was rejected by ramp buddy Adam Wolf because he wrote it by hand — Wolf told him to use the internal "Clyde" tool instead
- Built [[Claude Code]] as a personal side project; sole developer for the first three months
- Now leads the Claude Code team, which includes engineers, designers, data scientists, and a product lead (Cat Woo)
- Spent time studying reinforcement learning at Anthropic to [[Understand the Layer Under|understand the layer under]] Claude Code

## AI-Native Workflow

Boris is one of the most documented practitioners of [[AI-Native Workflow]]:

- Ships **20–30 PRs per day** with zero hand-written code and zero manual line edits
- Uses five terminal tabs, each with a separate repo checkout, round-robining between parallel Claude Code sessions
- Starts almost every session in **plan mode**, focusing on getting the plan right before implementation
- Overflows to the Claude desktop and iOS apps — roughly a third to half of code written from his phone
- **Uninstalled his IDE** after Opus 4.5 shipped
- During a December "coding vacation" traveling through Europe, Opus 4.5 wrote 100% of every PR with only ~2 bugs vs. an estimated ~20 if hand-written

> "I'll be honest, it writes better code than I do." — Boris Cherny

## Views

- Compares the current AI moment to [[The Printing Press Analogy|the printing press]]: software engineers are modern scribes; AI democratizes the skill
- Advocates for [[Generalist vs Specialist|generalists]] over specialists — believes the next trillion-dollar startup might be built by one person
- Sees strong opinions about languages and frameworks as increasingly irrelevant: "the model can just rewrite it for you"
- Values intellectual humility and beginner's mindset — "I'm wrong like half the time"
- Champions [[Prototyping Over PRDs]]: "we don't really write stuff — we just show"
- Believes safety is "the most important thing" — frames it as a [[Swiss Cheese Model (Safety)|Swiss cheese model]] with multiple overlapping layers
