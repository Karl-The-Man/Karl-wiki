# Steve Yegge

Software engineer and writer. Cited by [[Garry Tan]] for the claim that people using AI coding agents are "10x to 100x as productive as engineers using Cursor and chat today, and roughly 1000x as productive as Googlers were back in 2005."

Tan uses this as the opening hook for [[Thin Harness, Fat Skills (Garry Tan)|"Thin Harness, Fat Skills"]], arguing the productivity gap Yegge describes is real but commonly misattributed to model intelligence. The actual cause, per Tan: architecture — specifically the [[Thin Harness, Fat Skills]] pattern.
