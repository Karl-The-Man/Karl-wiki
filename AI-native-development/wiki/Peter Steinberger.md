# Peter Steinberger

Developer famous for his parallelized coding agent workflow. Known for a photo of himself in front of a monitor tiled with many [[Codex]] agents, each taking ~20 minutes on high-effort tasks across multiple repo checkouts. Creator of [[OpenClaw]].

## Workflow

- Tiles multiple [[Codex]] agents on a single monitor
- Each agent runs for ~20 minutes on high-effort prompts
- Multiple repos checked out simultaneously
- Cycles between agents, assigning work — the archetype of [[Macro Actions]]

[[Andrej Karpathy]] describes him as the person "you want to be" when it comes to agent-first development.

## OpenClaw

Created [[OpenClaw]], which [[Andrej Karpathy|Karpathy]] credits with innovating simultaneously in multiple dimensions:

1. **Soul and demeanor document** — a crafted personality that is "compelling and interesting"
2. **Sophisticated memory system** — beyond default context compaction
3. **WhatsApp portal** — single interface to all automation
4. **Personality matters** — Karpathy praises this as something "a lot of the current agents don't get correctly"
